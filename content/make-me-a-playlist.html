Make Me a Playlsit

Quick Facts

Stack: Javascript, Handlebars, Spotify API

Time frame: January 2015

Final Product: Website

Source Code: Playlist generator

Summary

In the cold of winter, I got tired of making new plalists by hand, so naturally, I created a web interface for making new playlists. Just select one of your playlists as the base, and create something new based off of it. I built it in an afternoon with some plain ol' JS and a little Handlebars to spice it up.

Now my only problem is having too much new music to listen to.
