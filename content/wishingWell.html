Wishing Well

Quick Facts

Languages & API: HTML, CSS, jQuery, Javascript, PHP, Venmo API

Time frame: March 2014

Collaborators: Luke Geiken, Alex Rinehart, Jake Long

Final Product: The Wishing Well

Source Code: Check out github!

Summary

For Iowa State's premier hackathon, HackISU, in Spring 2014, my team and I decided to utilize the Venmo API to create a web page which would serve as a social experiment.

Inspired by the concept of a wishing well, where passerbys throw money into a public space without even the thought of stealing it, we decided to create an online wishing well. Users have the ability to both throw money into the well and grab out a piece of 'change.'

On this web app I focused on both the high level design and front-end design of the website. I also worked closely with the Venmo API to log in and send requests.

The demo app has functionality turned off so don't worry about accidentally spending money.
