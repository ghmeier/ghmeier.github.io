This Website

Not only are all of the projects off to the side my work, but this website is all my work as well.

Everything you see here is coded from scratch using solely HTML, CSS, Javascript, and jQuery. The only thing generated from tools was the html for the center content which was parsed from markdown and the icons interspersed throughout. I decided to forgo some of the more powerful tools for web development in favor of doing things with just the basics because I feel that it provides the most opportunity for learning.

Check this out on Github!
